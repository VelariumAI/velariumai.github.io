# Todd Eddings - Personal Homepage

Clean, professional homepage for Todd Eddings showcasing VelariumAI, VRM-AI, and selected work.

## Overview

Single-page static site built for GitHub Pages. No frameworks, minimal dependencies, fast loading.

## Structure

- `index.html` - Main homepage
- `styles.css` - Responsive CSS styling
- `site-README.md` - This file

## Local Development

Open `index.html` directly in any web browser:

```bash
# Simple local server (optional)
python -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment

Ready for GitHub Pages deployment. Simply:

1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Set source to main branch

## Features

- Mobile responsive design
- Semantic HTML structure
- Professional typography
- Single accent color (#2563eb)
- Fast loading (no heavy dependencies)
- Accessible contrast ratios
